서버구축및운영실습

본 과목에서는 서버로 사용되는 운영체제인 리눅스의 구조 및 특징, 활용 방법 등에 대하여 학습한다.
이를 위하여 리눅스의 설치 방법, 명령어 사용법, 시스템 및 서버 관리 방법까지 이론과 실습을 병행함으로써 운영체제의 동작에 대해 이해하고 활용할 수 있는 능력을 키운다.

출제 범위 | 서버구축및운영실습 1~6주차 진도
Q1

리눅스 시스템의 구조 중 하드웨어를 제어하고 운영체제의 핵심 역할을 수행하는 것은 무엇입니까?

Q2

현재 리눅스의 표준 쉘로 사용되며, 방향키 기능을 지원하고 본 쉘(sh)과 호환성이 높은 쉘의 이름은 무엇입니까? (주관식)

Q3

관리자 계정(root)으로 로그인했을 때 터미널에 표시되는 프롬프트 기호는 무엇입니까?

Q4

mkdir 명령어로 'a/b/c'와 같이 하위 디렉토리까지 한 번에 생성하려고 할 때 사용하는 옵션은 무엇입니까?

Q5

파일이나 디렉토리를 삭제하는 rm 명령어에서, 비어 있지 않은 디렉토리를 강제로 삭제하기 위해 사용하는 옵션 조합은 무엇입니까? (주관식)

Q6

파일의 수정 시간을 현재 시간으로 변경하거나 내용이 없는 빈 파일을 생성할 때 사용하는 명령어는 무엇입니까? (주관식)

Q7

ls -l 실행 결과에서 파일 유형이 'l'로 표시된다면 이는 무엇을 의미합니까?

Q8

파일의 권한이 rwxr-xr-x일 때, 이를 8진수 숫자로 변환하면 무엇입니까? (주관식)

Q9

다음 중 윈도우의 '바로가기'와 유사하며 원본 파일의 경로 정보를 담고 있는 링크 방식은 무엇입니까?

Q10

명령어의 결과값을 기존 파일의 내용 뒤에 이어서 추가(append)하려고 할 때 사용하는 리다이렉션 기호는 무엇입니까? (주관식)

Q11

vi 편집기의 명령 모드에서 현재 커서가 위치한 한 줄 전체를 삭제(오려두기)하는 명령키는 무엇입니까?

Q12

vi 편집기에서 수정된 내용을 저장하고 강제로 종료하기 위해 사용하는 콜론(:) 명령어는 무엇입니까? (주관식)

Q13

find 명령어를 사용하여 현재 디렉토리 이하에서 파일 이름이 'test'로 시작하는 모든 파일을 찾는 명령어 형식으로 옳은 것은?

Q14

grep 명령어를 사용할 때 대소문자를 구분하지 않고 패턴을 검색하도록 설정하는 옵션은 무엇입니까? (주관식)

Q15

실행 중인 프로세스의 상태를 확인할 때, 백그라운드 프로세스를 포함하여 전체 시스템 프로세스를 자세히 보기 위해 사용하는 옵션 조합은 무엇입니까?
